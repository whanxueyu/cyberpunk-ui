## button 按钮
这是一个最基础的按钮，也是最具代表性的
<cp-button>CyberPunkUI</cp-button>

```vue
<template>
    <cp-button>CyberPunkUI</cp-button>
</template>
```

| 名称        | 类型           | 默认值  | 说明  |
| ------------- |:-------------:| -----:| -----:|
| color      | string | #333333 | 支持色值#333333或red |
| size      | number      |   14 |  支持屏幕缩放 |
| icon | string      |    AddCircle |  参考图标库 | 作者：MissBluee https://www.bilibili.com/read/cv27664516/ 出处：bilibili
